Files in this directory include:

dwca-uwymv_herp.zip - a Darwin Core archive with eight occurrence records downloaded from an instance of an Integrated Publishing Toolkit. Source: http://ipt.vertnet.org:8080/ipt/resource.do?r=uwymv_herp accessed 28 Jul 2015.
eight_specimen_records.csv - a CSV sample Simple Darwin Core occurrence file of eight records.
GBIFLemurs.zip - an archive of 4783 occurrences downloaded from the results of a query for scientific name 'Primates' and country 'Madagascar on the GBIF portal. Source: http://www.gbif.org/occurrence/search?TAXON_KEY=798&COUNTRY=MG, accessed 10 Aug 2015.
